In JavaScript, you can use the Array.includes() method to check if an array contains a specific value. Here's an example:

javascript
Copy code
const fruits = ["Apple", "Banana", "Orange"];
const targetValue = "Apple";

const result = fruits.includes(targetValue);

console.log(result); // true
In this example, the Array.includes() method returns true if the fruits array contains the targetValue, which is "Apple". If the fruits array does not contain the targetValue, Array.includes() returns false.
